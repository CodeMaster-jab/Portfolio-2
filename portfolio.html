Interactive Job Search
Alek Seawright, Avery Rouser, Jeffrey Bell

This applicaiton was built using the Foundations CSS3 Framework, and consumes two API calls to Job Services from USAJOBS and THEMUSE. The user can enter a Location (City or Zip Code), and either select a Job Category, or enter a Search Keyword. Then the Jobs found for the given Search criteria will be displayed to the user.
